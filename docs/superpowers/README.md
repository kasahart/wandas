# Superpowers Working Documents

This directory is only for short-lived agent-created working artifacts while repository work is in progress.

Retention policy:

- Long-term architecture decisions belong in `docs/design/`.
- User-facing architecture and how-to material belongs under `docs/src/`.
- Completed specs and implementation plans should be deleted after their durable decisions have been moved into `docs/design/` or `docs/src/`.
- `plans/` is for temporary step-by-step execution plans. Completed plans should not be committed; `.gitignore` ignores this directory for new files.

Do not keep duplicate historical specs here once the implementation has landed and the important contract is documented elsewhere.
