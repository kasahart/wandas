# Superpowers Working Documents

This directory holds agent-created design artifacts while repository work is in
progress.

Retention policy:

- `specs/` may keep short design records when they explain an active or recently
  settled implementation contract.
- `plans/` is for temporary step-by-step execution plans. Completed plans should
  not be committed; `.gitignore` ignores this directory for new files.
- Long-term architecture decisions belong in `docs/design/`.

When a temporary plan has already landed, remove it instead of preserving the
implementation log as durable documentation.
