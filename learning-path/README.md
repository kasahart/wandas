# Wandas 学習パス

## 信号処理・音響解析のための包括的なガイド

この学習パスは、Wandasを使って信号処理・音響解析を始めるための**統合されたステップバイステップガイド**です。チュートリアルと実践例を統合し、ユーザーがWandasを使う**動機**と**実践的なスキル**を同時に身につけられるように設計されています。

## 🎯 この学習パスの目的

**信号処理の初心者から上級者まで**、Wandasを使って以下を達成できるようにします：

- ✅ **なぜWandasを使うのか**を理解する
- ✅ **基本的な信号処理**をpandasライクなAPIで実装
- ✅ **大規模データ**を効率的に処理
- ✅ **美しい可視化**を作成
- ✅ **実世界の問題**を解決

## 📚 学習の流れ

各ノートブックは**「問題提起 → Wandasでの解決 → 実践例」**のストーリーで構成されています。

### 1. [00_why_wandas.ipynb](00_why_wandas.ipynb) - Wandasとは何か

**なぜWandasが必要なのか？**

- 信号処理の課題と従来のアプローチの問題点
- Wandasの特徴と利点
- どんな問題を解決できるか

### 2. [01_getting_started.ipynb](01_getting_started.ipynb) - 環境構築とウォームアップ

**Wandasを動かしてみよう**

- インストールと環境設定
- 最初の信号生成と可視化
- Jupyter環境でのインタラクティブな操作

### 3. [02_working_with_data.ipynb](02_working_with_data.ipynb) - データの読み込みと操作

**現実のデータをWandasで扱う**

- WAV/CSVファイルの読み込み
- ChannelFrameデータ構造の理解
- データのアクセスと基本操作

### 4. [03_signal_processing_basics.ipynb](03_signal_processing_basics.ipynb) - 信号処理の基礎

**周波数分析とフィルタリング**

- FFTによる周波数領域変換
- ローパス/ハイパス/バンドパスフィルター
- 実践的なオーディオ処理例

### 5. [04_advanced_signal_processing.ipynb](04_advanced_signal_processing.ipynb) - 高度な信号処理

**スペクトログラムと時間周波数分析**

- STFTとスペクトログラム
- 時間周波数解析の応用
- 信号の特徴抽出

### 6. [05_visualization.ipynb](05_visualization.ipynb) - データの可視化

**結果を効果的に伝える**

- 波形・スペクトル・スペクトログラムの可視化
- カスタムプロットとインタラクティブ機能
- 出版品質のグラフ作成

### 7. [06_real_world_applications.ipynb](06_real_world_applications.ipynb) - 現実世界の応用

Wandasを実際のデータ分析に活用

- 音声信号処理（品質評価）
- 振動分析（機械診断）
- 生体信号処理（HRV分析）
- クロスドメイン統合分析

### 8. [07_conclusion_and_next_steps.ipynb](07_conclusion_and_next_steps.ipynb) - まとめと次のステップ

学習の振り返りと今後の学習計画

- これまでの学習内容の振り返り
- 自分の理解度の評価
- 今後の学習ロードマップ
- プロジェクトアイデアと実践的な提案

## 🚀 学習を始める前に

### 必要な環境

- Python 3.9+
- Jupyter Notebook/Lab
- 基本的な信号処理の知識（なくてもOK）

### 推奨される学習順序

1. **初心者**: 00 → 01 → 02 → 03 → 05
2. **信号処理経験者**: 00 → 02 → 03 → 04 → 06
3. **データサイエンティスト**: 00 → 01 → 06 → 07

## 💡 学習のポイント

### 🎯 動機付け重視

各ノートブックで**「なぜこの機能が必要か」**を明確に説明。単なる機能紹介ではなく、**問題解決の文脈**で学ぶことができます。

### 🔄 実践重視

理論だけでなく、**すぐに使えるコード例**を豊富に用意。コピー&ペーストで試せます。

### 📈 段階的難易度

基本から応用まで**自然な学習曲線**。挫折せずに上達できます。

### 🎨 可視化中心

信号処理の結果を**直感的に理解**できる美しいグラフを重視。
